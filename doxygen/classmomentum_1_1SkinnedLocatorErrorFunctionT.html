Detailed Description

template<typename T>
class momentum::SkinnedLocatorErrorFunctionT< T >

Error function for skinned locator constraints.

This error function computes the error between skinned locator positions and their target positions. Skinned locators are points that are deformed by the character's skeleton and can be used to constrain the character to specific world positions. The error is computed as the weighted squared distance between the current skinned locator position and the target position.
