WeightMatrix momentum::SkinWeights::weight

Weight of each joint's influence on each vertex.

Each row corresponds to a vertex, and each column contains the weight of a joint's influence on that vertex. Weights for a vertex typically sum to 1.0. Unused influences are set to 0.0.
