Detailed Description

A highly optimized error function for 3d position errors.

This class is best used when dealing with a large number of constraints. For fewer constraints, consider using the generic PositionErrorFunction instead.

Warning
Due to the multi-threaded evaluation of the error/gradient, the functions are non-deterministic, which may cause numerical inconsistencies. As a result, slightly different results may be produced on multiple calls with the same data.
