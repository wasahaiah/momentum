Detailed Description

template<typename T>
struct momentum::CharacterStateT< T >

Represents the complete state of a character at a specific point in time.

Stores the character's parameters, skeleton state, locator positions, skinned mesh, and collision geometry state. This class is used for character animation and rendering.
